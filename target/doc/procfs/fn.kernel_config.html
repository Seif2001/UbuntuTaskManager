pub fn kernel_config() -> ProcResult<HashMap<String, ConfigSetting>>
Expand description

Returns a configuration options used to build the currently running kernel

If CONFIG_KCONFIG_PROC is available, the config is read from /proc/config.gz. Else look in /boot/config-$(uname -r) or /boot/config (in that order).

Notes
